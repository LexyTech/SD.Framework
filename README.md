#项目概述

1、ShSoft.Framework2016.Common
	
	公共基础类库，包含一些常用扩展工具。


2、ShSoft.Framework2016.Infrastructure

	基础设施，包含系统常量、自定义异常、基接口与基类。


3、ShSoft.Framework2016.Infrastructure.AOP

	AOP基础设施，包含为不同项目封装的不同处理异常的方式。


4、ShSoft.Framework2016.Infrastructure.DomainEvent

	领域事件基础设施，包含领域事件基类、领域事件处理者工厂、领域事件中介者、Session存储提供者。


5、ShSoft.Framework2016.Infrastructure.DomainEvent.EFStorer

	EntityFramework领域事件存储提供者。


6、ShSoft.Framework2016.Infrastructure.Global

	全局操作基础设施，包括全局初始化、释放资源、事务。


7、ShSoft.Framework2016.Infrastructure.MVC.IOC

	MVC客户端依赖注入基础设施。


8、ShSoft.Framework2016.Infrastructure.Repository

	仓储基础设施，包含EntityFramework DbContext的三次封装、数据库清理者、EntityFramework仓储提供者、EntityFramework UnitOfWork提供者。


9、ShSoft.Framework2016.Infrastructure.WCF.IOC

	WCF服务端依赖注入基础设施。


10、ShSoft.Framework2016.Infrastructure.WebApi

	WebApi客户端依赖注入基础设施。
